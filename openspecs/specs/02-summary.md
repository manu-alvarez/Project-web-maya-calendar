# Summary

## Project Overview

Create a web platform that integrates ancient Maya calendar wisdom with modern web technology, providing users with access to information about the Maya synchronary calendar.

## Vision

This project is not just a digital product, but a platform for cultural, educational, and symbolic transmission. The code is the channel, and purpose is the compass.

## Core Purpose

To bridge the gap between ancient Maya wisdom and modern accessibility by:

- Providing accurate, professional quality information about the Maya synchronary calendar
- Making this knowledge accessible to both scholars and casual users
- Creating a platform that can evolve and scale with future needs
- Maintaining cultural integrity while leveraging modern technology

## Key Features

1. **Daily Kin Consultation**: Users can view today's Kin or any specific date
2. **Encanted Wave Calculation**: Calculate and display the 13-Kin Encanted Wave for any date
3. **User Management**: Anonymous and registered user access with different privilege levels
4. **Date Management**: Save important dates and set reminders (registered users)
5. **Authentication**: Email with OTP verification and Google OAuth

## Target Users

### Anonymous/Casual Users
- Quick consultations without registration
- Access to basic Kin and Onda Encantada information

### Registered Users
- Save important dates
- Set reminders for specific dates
- Access extended content

### Scholars
- In-depth study of Maya calendar
- Detailed interpretations
- Comprehensive data access

## Value Proposition

- **Professional Quality**: Rigorous, validated content
- **Didactic Approach**: Educational and accessible information
- **Comprehensive Content**: Equivalent to A4 page descriptions
- **Visual Excellence**: Professional illustrations and design
- **Real Data**: 100% production environment, no mockups
