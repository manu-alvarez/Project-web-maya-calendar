# Project Context

## Vision

Create a web platform that integrates ancient Maya calendar wisdom with modern web technology, providing users with access to information about the Maya synchronary calendar, including:

- Daily Kin calculations
- Encanted Wave (Onda Encantada) calculations
- Date-specific consultations
- Personal interpretations and applications

## Purpose

This project is not just a digital product, but a platform for cultural, educational, and symbolic transmission. The code is the channel, and the purpose is the compass.

## Target Users

### Primary User Types

1. **Anonymous/Casual Users**
   - Access basic Kin information
   - View daily Kin
   - Consult specific dates
   - Browse general information

2. **Registered Users**
   - All anonymous user features
   - Save important dates
   - Set reminders for specific dates
   - Access extended content
   - Personalized experience

3. **Scholarly Users**
   - In-depth study of Maya calendar
   - Detailed interpretations
   - Research capabilities
   - Comprehensive data access

## Value Proposition

### Core Value

- **Professional Quality**: Rigorous, validated content
- **Didactic Approach**: Educational and accessible information
- **Comprehensive Content**: Equivalent to A4 page descriptions
- **Visual Excellence**: Professional illustrations and design
- **Real Data**: 100% production environment, no mockups

## Product Characteristics

### Content Standards

- Kin descriptions: Full A4 page equivalent with detailed explanations
- Multiple illustrations where appropriate

### User Experience

- Simple and intuitive interface
